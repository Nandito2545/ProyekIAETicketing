# Event-Ticket Service

Microservice untuk mengelola Event dan Ticket menggunakan gRPC.

## Setup

1. Install dependencies
```bash
   npm install
```

2. Setup environment
```bash
   cp .env.example .env
   # Edit .env sesuai konfigurasi
```

3. Run service
```bash
   npm run dev
```

## Service Port
- gRPC: 50052

## Dependencies
- @grpc/grpc-js
- mongoose
- dotenv
